Cycle time = data collection CT + image post-processing CT；估算以收圖機時作主要認列，後處理、repeat scan 與 decision CT 改善作為保守 buffer。
4,500 萬 + 5,400 萬 = 9,900 萬

Problem Statement

目前 FA 高解析影像受 noise、vibration 與局部 distortion 影響，導致 morphology 特徵判讀困難、3D reconstruction 品質不穩定；為取得可用影像需提高 dwell time、repeat scan 與人工後處理，使 end-to-end image CT 拉長，影響 FA 時效與 CIP 方向判斷。

WhereFA high-resolution imaging、3D reconstruction 前處理流程
When低 dwell time、長時間收圖、對 vibration 敏感條件下特別明顯
WhoFA engineer、3D reconstruction user、CIP decision owner
Why影像品質不足會放大判讀不確定性，導致改善方向錯誤風險
How透過拉長 dwell time / repeat scan 補償品質，但犧牲 tool time
How muchdata collection 約 3 天 + 待補後處理時間；改善目標：end-to-end image CT 降低 60–70%

D3｜執行、驗證暫時防堵措施

永久對策完成前，先用傳統 filter 與現場改善降低影像品質風險

品質工具：Containment Plan / Risk Matrix / FMEA-lite

暫時防堵措施｜先用既有方法降低風險

  • 傳統 filter：Gaussian / Median / NLM 先行降噪，作為暫時影像改善方法
  • 現場操作：鎖緊 holder、確認 sample mounting、必要時鍍 Pt 降低 charging / artifact
  • 環境控制：收圖期間降低人員走動與講話聲量，減少 vibration / acoustic disturbance
  • 判讀保護：filter 後影像只作輔助判讀，critical morphology 仍需 raw image / repeat scan 確認
  • QC gate：進入 3D reconstruction 前先檢查 noise、edge blur 與 alignment feasibility

風險評估與防堵驗證

風險項目SODRPN暫時防堵
noise 導致 morphology 誤判865240Gaussian / Median / NLM + raw image 比對
vibration 導致阻障層假缺陷955225鎖緊 holder、降低走動 / 講話、repeat scan
charging / contamination 造成 artifact756210sample mounting 確認 / 必要時鍍 Pt

驗證方式

比較 raw、Gaussian、Median、NLM 與現場改善後影像的 noise index、edge blur、morphology 判讀一致性。若 filter 造成特徵模糊或仍低於 QC gate，需回到 raw image、repeat scan 或重新固定 sample，不直接進入 final conclusion。

Filter containment

Gaussian / Median / NLM 可快速降低 noise，但需檢查 edge blur 與 morphology distortion。

現場 containment

鎖 holder、鍍 Pt、降低環境擾動，先把 vibration / charging 風險壓低。

退出條件

永久對策完成後，以 AI restoration + image QC gate 取代臨時 filter 與人工經驗判斷。

5 Why 驗證真因

1 原因影像有 noise、vibration artifact、局部 distortion。
